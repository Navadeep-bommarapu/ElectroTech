# ElectroTech - Digital Product Catalogue

A modern, responsive digital catalogue website for an electronics product company. Built with React, Vite, and Tailwind CSS.

## üöÄ Overview

ElectroTech is a premium product showcase platform designed to display electronic products in a sleek, professional manner. **Note:** This is a catalogue website, not an e-commerce store. It focuses on product discovery, specifications, and brand storytelling without cart or checkout functionality.

## ‚ú® Features

- **Modern UI/UX**: Clean, minimalist design with a Royal Blue theme and glassmorphism effects.
- **Product Showcase**:
  - **Featured Products**: Highlighted top-tier items.
  - **Category Browsing**: Filter products by Laptops, Phones, Watches, etc.
  - **Product Details**: Deep dive into specs, images, and features.
- **Digital Catalogue**: dedicated page for downloading product brochures and datasheets.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Interactive Elements**: Smooth hover effects, sticky navigation, and animated transitions.

## üì¶ Installation

Cloning the repository and running the project locally:

```bash
# Clone the repository
git clone https://github.com/Navadeep-bommarapu/Digital-Catalogue.git

# Navigate into the directory
cd Digital-Catalogue

# Install dependencies
npm install

# Start the development server
npm run dev
```

## üöÄ Deployment

This project is optimized for deployment on Vercel.

1. Push your code to GitHub.
2. Import the repository in Vercel.
3. Vercel will auto-detect the Vite settings.
4. Click **Deploy**.

*A `vercel.json` file is included to handle Single Page Application (SPA) routing.*
